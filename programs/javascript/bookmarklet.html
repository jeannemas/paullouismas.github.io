What is a bookmarklet?

A bookmarklet is a bookmark stored in a web browser that contains JavaScript commands that add new features to the browser. (Wikipedia)

How to install a bookmarklet?

You can drag and drop the link that contain the bookmarklet onto your bookmark bar, or alternatively, copy the code of the bookmarklet as the URL of a new bookmarklet.
